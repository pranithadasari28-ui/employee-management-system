# Employee Management System

A GUI-based Employee Management System built using Python, Tkinter, and MySQL. It performs CRUD operations like Add, View, Update, and Delete employee records.

## Features
- Add Employee  
- View Employees  
- Update Employee  
- Delete Employee  
- MySQL Database Integration  
- Tkinter GUI Interface  

## Technologies Used
- Python  
- MySQL  
- Tkinter  

## How to Run

1. Install required package:
   pip install mysql-connector-python

2. Make sure MySQL server is running and database is created:
   employee_db

3. Run the program:
   python employee_gui.py
